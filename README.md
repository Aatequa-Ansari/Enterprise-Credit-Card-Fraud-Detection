## 📖 Project Overview

Credit card fraud represents one of the most persistent threats within global financial ecosystems. In streaming real-time transactional setups, fraudulent attempts compose only a microscopic fraction of the total clearing ledger. This severe class imbalance easily blinds standard modeling setups and requires industrial-grade preprocessing, robust resampling frameworks, and complex tree/neural network ensembles.

This system delivers an end-to-end framework capable of ingesting raw anonymized transactional tensors and delivering low-latency inference profiles with high-precision certainty scores. By exposing model internals through SHAP feature attributions, the platform translates abstract statistical values into digestible decision chains optimized for fraud risk teams.

### Core Core Capabilities:
* **Inference Pipelines:** Provides instant risk score assessment via manual form fields or asynchronous batch processing using `.csv` uploads.
* **Model Comparison Matrix:** Live benchmarking comparisons sorting 15 machine learning and 5 deep learning topologies across precision, recall, and time dimensions.
* **Explainable ML Routing:** Global feature attribution layers that expose the mathematical rationale driving the target classification outputs.

## ✨ Key Features & Technical Depth

### 🔍 Risk Inference
* Real-time generation of transaction risk levels (Low, Medium, High).
* Dynamic adjustment vectors via optional mathematical preprocessing scalers.
* Asynchronous background processing framework for batch data processing.

### 🤖 Machine Learning Ensembles
* **Linear & Baseline Classifiers:** Logistic Regression, SGD Classifier, Passive Aggressive.
* **Tree/Boosting Topologies:** Random Forest, Extra Trees, Gradient Boosting, Histogram Gradient Boosting, AdaBoost, LightGBM, CatBoost, and XGBoost.
* **Hyperparameter Tuning Configurations:** Grid/Randomized hyperparameter sweeps mapping out tuned variations of high-performing frameworks.

### 🧠 Deep Learning Architectures
* **Supervised Classifiers:** Feed-Forward Multi-Layer Artificial Neural Networks (ANN) & Deep Neural Networks (DNN) featuring explicit Dropout regularizations.
* **Unsupervised Architectures:** Bottleneck Reconstruction Autoencoders identifying anomalous fraud profiles through Reconstruction Loss Variance anomalies.

### 💡 Model Interpretability & Explainability
* Mathematical tracking of structural contributions across latent components ($V_1$ to $V_{28}$).
* Global feature tracking maps outlining system vulnerabilities.

## 🔄 Core Pipeline Execution Sequence

### 1. Preprocessing Engine

* **Null Deletion & Outlier Scans:** Complete check verifying zero missing fields across the feature array.
* **Variance Standardization:** Optional mapping via Standard, MinMax, or Robust Scalers to clean continuous transaction distributions prior to pipeline routing.

### 2. Class Imbalance Engineering

To resolve the 0.172% minority skew, the pipeline supports a collection of advanced sampling techniques:

* Synthetic Minority Over-sampling Technique (**SMOTE**)
* Advanced Boundary Focus Variants (**Borderline-SMOTE**, **ADASYN**)
* Hybrid Deletion-Over-sampling Structs (**SMOTE-Tomek**, **SMOTE-ENN**)

### 3. Hyperparameter Tuning Methods

* Automated Grid & Randomized Cross-Validation runs executing over hyperparameter depths for Extra Trees and XGBoost.
* Weight Adjustments within neural network training scripts to penalize misclassified minority samples.

## 🚀 Local Installation & Execution

Follow these steps to set up and run the **Enterprise Credit Card Fraud Detection System** on your local machine.

### 1. Clone the Repository

```bash
git clone [https://github.com/Aatequa-Ansari/Enterprise-Credit-Card-Fraud-Detection.git](https://github.com/Aatequa-Ansari/Enterprise-Credit-Card-Fraud-Detection.git)
cd Enterprise-Credit-Card-Fraud-Detection

'''

### 2. Create and Activate a Virtual Environment

#### Create Virtual Environment

```bash
python -m venv venv
```

#### Activate on Windows

```bash
venv\Scripts\activate
```

#### Activate on Linux / macOS

```bash
source venv/bin/activate
```

---

### 3. Install Required Dependencies

Upgrade pip and install all project dependencies.

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

---

### 4. Launch the Streamlit Application

```bash

```
